🧮 Calculadora Profissional

Uma calculadora moderna, responsiva e segura desenvolvida utilizando apenas HTML5, CSS3 e JavaScript Vanilla, sem bibliotecas ou frameworks.

O projeto foi criado com foco em boas práticas de desenvolvimento Front-end, código organizado, experiência do usuário e manipulação segura de expressões matemáticas.

🛡️ Segurança

Embora seja uma calculadora simples, foi implementada uma camada de proteção para evitar execução de código arbitrário.

Antes de calcular uma expressão, ela passa por um processo de sanitização, permitindo apenas:

- números
- ponto decimal
- operadores matemáticos (+ - * /)

Isso impede que códigos externos sejam executados durante o cálculo.
